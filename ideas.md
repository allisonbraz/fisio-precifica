## Abordagem 1: "Clinical Precision" — Design Suíço Médico

**Design Movement**: Swiss/International Typographic Style com influências de design médico-clínico

**Core Principles**:
1. Precisão tipográfica — hierarquia clara e legibilidade absoluta
2. Grid rígido com espaços brancos generosos — dados financeiros precisam de respiro
3. Monocromatismo funcional com acentos cirúrgicos — cor usada com intenção, não decoração
4. Confiança institucional — transmitir credibilidade profissional

**Color Philosophy**: Base em branco puro (#FAFBFC) com cinza grafite (#1A1D23) para texto. Acento primário em verde-azulado médico (#0D9488 — teal-600) que remete a saúde sem ser clichê. Acento secundário em âmbar (#F59E0B) para alertas e destaques financeiros. Vermelho (#EF4444) apenas para valores negativos/prejuízo.

**Layout Paradigm**: Sidebar fixa à esquerda com navegação vertical. Área principal em grid de 12 colunas. Cards com bordas sutis e sombras mínimas. Tabelas como elemento central de apresentação de dados.

**Signature Elements**: 
- Números financeiros com fonte monospace para alinhamento perfeito
- Indicadores de status com dots coloridos (verde/amarelo/vermelho)
- Micro-gráficos sparkline inline nos cards de resumo

**Interaction Philosophy**: Transições suaves de 200ms. Feedback imediato em inputs numéricos. Tooltips informativos com fórmulas de cálculo.

**Animation**: Entrada de cards com fade-in sutil. Números que "contam" ao mudar de valor. Barras de progresso animadas nos indicadores.

**Typography System**: DM Sans para headings (peso 600-700), Inter para body text (400-500), JetBrains Mono para valores monetários.

## Abordagem 2: "Warm Professional" — Design Orgânico Contemporâneo

**Design Movement**: Organic Modernism com toques de design escandinavo

**Core Principles**:
1. Calor humano — fisioterapia é cuidado, o app deve refletir isso
2. Formas suaves e orgânicas — cantos arredondados generosos, ondulações sutis
3. Dados acessíveis — transformar números em narrativas visuais compreensíveis
4. Progressão natural — guiar o usuário como um fluxo terapêutico

**Color Philosophy**: Base em creme quente (#FBF8F4) que acolhe sem cansar. Primário em terracota suave (#C2785C) que transmite calor e profissionalismo. Secundário em verde sage (#7C9A82) para indicadores positivos e saúde. Texto em marrom escuro (#3D2E2E) para suavidade. Acentos em dourado (#D4A853) para destaques premium.

**Layout Paradigm**: Layout fluido com seções empilhadas verticalmente no mobile. Desktop com sidebar colapsável. Seções separadas por divisores orgânicos (ondulações SVG). Cards com cantos generosos (16-20px radius) e sombras difusas.

**Signature Elements**:
- Divisores ondulados entre seções
- Ícones ilustrativos hand-drawn style
- Gradientes suaves nos headers de seção

**Interaction Philosophy**: Hover states com escala sutil (1.02). Inputs com labels flutuantes. Feedback tátil — elementos "respiram" ao interagir.

**Animation**: Scroll-triggered reveals com ease-out. Gráficos que se desenham progressivamente. Transições de página com crossfade suave.

**Typography System**: Outfit para headings (peso 500-700), Source Sans 3 para body (400-500), Fira Code para valores monetários.

## Abordagem 3: "Dashboard Noir" — Design de Fintech Escuro

**Design Movement**: Dark UI Fintech com influências de design de trading platforms

**Core Principles**:
1. Dados como protagonista — interface escura faz números brilharem
2. Densidade informacional — mostrar mais dados sem sobrecarregar
3. Profissionalismo financeiro — transmitir seriedade sobre dinheiro
4. Controle total — o usuário se sente no comando dos seus números

**Color Philosophy**: Base em azul-escuro profundo (#0F172A) com superfícies em (#1E293B). Primário em cyan elétrico (#06B6D4) para ações e destaques. Verde neon (#10B981) para lucro/positivo. Vermelho coral (#F43F5E) para prejuízo/negativo. Âmbar (#FBBF24) para alertas. Texto em cinza claro (#E2E8F0).

**Layout Paradigm**: Dashboard full-width sem sidebar — navegação por tabs no topo. Grid de cards com glassmorphism sutil. Gráficos grandes e interativos como peça central. Layout assimétrico com cards de tamanhos variados.

**Signature Elements**:
- Glassmorphism nos cards principais
- Glow effects sutis nos valores importantes
- Gradientes de borda nos cards ativos

**Interaction Philosophy**: Hover com glow e elevação. Drag para reorganizar cards. Inputs com feedback visual instantâneo (cores mudam conforme valor).

**Animation**: Números com animação de contagem. Gráficos com transições fluidas. Partículas sutis no background do header.

**Typography System**: Space Grotesk para headings (peso 500-700), Geist Sans para body (400-500), Geist Mono para valores monetários.

## Decisão: Abordagem 2 — "Warm Professional" (Organic Modernism)

Escolho a Abordagem 2 porque:
1. Fisioterapia é uma profissão de cuidado humano — o design deve refletir isso
2. Muitos fisioterapeutas não são tech-savvy — a interface precisa ser acolhedora
3. O tom orgânico diferencia de calculadoras financeiras genéricas
4. As cores quentes criam confiança e reduzem a ansiedade ao lidar com finanças
5. O layout fluido funciona bem em mobile (muitos fisioterapeutas usam celular)
