# Story 1.2 — Consistência dos Níveis de Lucro em Todas as Páginas

**Epic:** Correção do Modelo de Precificação
**Status:** Ready
**Prioridade:** CRÍTICA
**Agente:** @dev
**Depende de:** Story 1.1

---

## Contexto

Após a Story 1.1 corrigir a fórmula, todas as páginas precisam exibir os 3 níveis de lucro de forma **consistente e com a mesma terminologia**:

```
Receita Bruta
  - Impostos sobre receita
= Receita Líquida
  - Custos Variáveis
= Margem de Contribuição
  - Custos Fixos (com pró-labore)
= Lucro Operacional
  - Reservas Estratégicas
= Lucro Disponível
```

### Problema Atual

| Página | Mostra | Problema |
|--------|--------|---------|
| Precificação | "Lucro Operacional" sem reservas | Número inflado |
| Simulação | "Lucro Operacional" sem reservas | Inconsistente com Home |
| Home | "Lucro Disponível" (com reservas) | Correto, mas diferente |
| Indicadores | "Lucro Disponível" (com reservas) | Correto |
| Reservas | "Lucro Disponível" (com reservas) | Correto |

O usuário vê números diferentes para "lucro" em páginas diferentes.

---

## Critérios de Aceite

### AC-1: Terminologia padronizada

Usar SEMPRE estas definições:

| Termo | Fórmula | Onde Mostrar |
|-------|---------|-------------|
| **Custo Total Mensal** | Fixos + Variáveis + Reservas | Todas as páginas |
| **Custo por Sessão** | Custo Total Mensal / sessoesMeta | Precificação, Home |
| **Receita Bruta** | Preço × sessoesMeta | Todas |
| **Impostos** | Receita × impostoPercentual | Precificação, Home |
| **Lucro Operacional** | Receita - Impostos - Fixos - Variáveis | Precificação, Simulação |
| **Lucro Disponível** | Lucro Operacional - Reservas | Home, Indicadores, Reservas |

- [ ] Padronizar labels em todas as páginas conforme tabela acima
- [ ] Nunca usar "Custo Mensal Total" (ordem invertida) — sempre "Custo Total Mensal"
- [ ] Nunca usar "Sessões Meta" sem explicar — usar "Meta de Sessões/Mês"

### AC-2: Home page mostra hierarquia completa
- [ ] Card principal: Receita Bruta estimada
- [ ] Card: Custo Total Mensal (fixos + variáveis + reservas)
- [ ] Card: Lucro Operacional
- [ ] Card: Lucro Disponível (destaque — é o número que importa)
- [ ] Tooltip em cada card explicando a fórmula

### AC-3: Precificação mostra decomposição do preço
- [ ] Barra horizontal mostrando: Custos Fixos | Custos Variáveis | Reservas | Impostos | Lucro
- [ ] Cada segmento com valor em R$ e % do preço
- [ ] Label clara: "Composição do Preço por Sessão"

### AC-4: Simulação mostra ambos os lucros
- [ ] Coluna "Lucro Operacional" nos cenários
- [ ] Coluna "Lucro Disponível" nos cenários (cor diferente, destaque)
- [ ] Gráfico atualizado com linha de Lucro Disponível

### AC-5: Indicadores usa Lucro Disponível consistentemente
- [ ] Score de margem baseado em Lucro Disponível / Receita (não Operacional)
- [ ] Label clara: "Margem Disponível" (não apenas "Margem")
