# Story 1.6 — Padronização de Terminologia e Eliminação de Redundâncias

**Epic:** Correção do Modelo de Precificação
**Status:** Ready
**Prioridade:** MÉDIA
**Agente:** @dev
**Depende de:** Story 1.2

---

## Contexto

Termos inconsistentes confundem o fisioterapeuta. O mesmo conceito aparece com nomes diferentes em páginas diferentes.

## Critérios de Aceite

### AC-1: Glossário único de termos
- [ ] Criar constantes de labels em `lib/labels.ts`:

```typescript
export const LABELS = {
  custoTotalMensal: 'Custo Total Mensal',
  custoPorSessao: 'Custo por Sessão',
  precoPorSessao: 'Preço por Sessão',
  receitaBruta: 'Receita Bruta Estimada',
  lucroOperacional: 'Lucro Operacional',
  lucroDisponivel: 'Lucro Disponível',
  sessoesMeta: 'Meta de Sessões/Mês',
  sessoesRealizadas: 'Sessões Realizadas',
  taxaOcupacao: 'Taxa de Ocupação',
  pontoEquilibrio: 'Ponto de Equilíbrio',
  valorHora: 'Valor por Hora',
  margemLucro: 'Margem de Lucro',
  reservasEstrategicas: 'Reservas Estratégicas',
} as const;
```

- [ ] Todas as páginas importam de `LABELS` em vez de strings hardcoded
- [ ] Eliminar variações: "Custo Mensal Total" → "Custo Total Mensal"

### AC-2: Hook `useMetrics()` compartilhado
- [ ] Criar `lib/useMetrics.ts` que centraliza todos os cálculos:

```typescript
export function useMetrics(data: DadosPrecificacao) {
  return useMemo(() => ({
    custoFixoMensal,
    custoVarMensal,
    reservasMensal,
    custoTotalMensal,
    custoPorSessao,
    precoPorSessao,
    receitaBruta,
    impostos,
    lucroOperacional,
    lucroDisponivel,
    taxaOcupacao,
    pontoEquilibrio,
    valorHora,
    capacidadeMaxima,
    // ...etc
  }), [data]);
}
```

- [ ] Home, Precificação, Simulação, Indicadores, Reservas usam `useMetrics()`
- [ ] Remover cálculos duplicados inline de cada página

### AC-3: Relatórios usa mesma terminologia
- [ ] Labels nos formulários de registro mensal alinhados com LABELS
- [ ] Gráficos usam mesmas cores para mesmos conceitos em todas as páginas
