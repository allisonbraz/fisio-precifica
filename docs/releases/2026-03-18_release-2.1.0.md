# Release 2.1.0 — UX Estratégico

**Data:** 2026-03-18
**Tema:** Landing page, onboarding guiado e reorganização da navegação

---

## Resumo

Release focada em **conversão e experiência do usuário**. Baseada em análise completa de UX e estratégia de produto, implementa as 5 melhorias de maior impacto para transformar visitantes em usuários ativos.

## Novas funcionalidades

### Landing Page com Calculadora Rápida
Visitantes sem login agora veem uma landing page profissional em vez do dashboard vazio. Inclui:
- Proposta de valor clara ("Descubra quanto você deveria cobrar por sessão")
- Seção "O problema" para gerar identificação
- **Calculadora Rápida** — 3 campos (custo mensal, sessões/mês, margem) geram resultado imediato sem criar conta
- Seção "Como funciona" em 3 passos visuais
- 6 cards de benefícios
- CTA final com badges de confiança

### Progresso Guiado no Dashboard
Substitui o wizard modal descartável por uma barra de progresso persistente com 4 etapas:
1. Cadastrar custos
2. Definir margem de lucro
3. Calcular preço por sessão
4. Criar tipos de serviço

Cada etapa é clicável e detecta automaticamente se foi concluída. Desaparece quando tudo está feito.

### Gate Progressivo
Páginas avançadas (Serviços, Simulação, Reservas, Relatórios) mostram banner contextual quando custos ou precificação não foram preenchidos. Orienta sem bloquear.

### Reservas como Aba em Custos
Reservas Estratégicas agora é a 4ª aba dentro de Custos (Operacionais / Depreciação / Variáveis / Reservas), eliminando uma página separada do menu.

## Melhorias de navegação

### Sidebar Reorganizado
De 10 itens sem hierarquia para 7 itens em 3 grupos:
- **Essencial:** Painel, Custos, Precificação, Indicadores
- **Ferramentas:** Serviços, Simulação, Relatórios
- **Rodapé:** Meu Perfil, Configurações

## Correções

- Texto "Seus dados ficam salvos localmente" corrigido para refletir sync com servidor
- Disclaimer "cálculos não incluem impostos" corrigido (app já inclui impostos)
- "Material descartável por sessão" renomeado para "Materiais de uso único por sessão"
